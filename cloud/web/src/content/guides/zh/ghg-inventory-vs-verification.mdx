这两个词在会议里经常被混着用，但不该混：**盘查**（碳盘查 / 温室气体清单
编制）是企业自己做的那本账；**核查**（碳核查）是独立第三方来检查这本账
是否实质准确。前者你可以自己干，后者按定义就不能自己干。混淆的代价是
真金白银 —— 要么买了没人要求的鉴证，要么到披露截止时才发现有人要求过。

## 一句话区别

> 企业（或其顾问）依据 ISO 14064-1 / GHG Protocol **编制**清单；
> 有资质的第三方机构依据 ISO 14064-3 **核查**清单并出具核查声明。

## 一张表对比

| | 碳盘查（清单编制） | 碳核查 |
| --- | --- | --- |
| 谁来做 | 企业自己，常委托咨询机构协助 | 独立的第三方核查机构 |
| 依据标准 | ISO 14064-1、GHG Protocol、主管部门核算指南 | ISO 14064-3；鉴证业务另有 ISAE 3410 |
| 输出 | 清单数据 + 盘查报告 | 核查声明 / 鉴证意见 |
| 时点 | 每年年度数据关账后 | 清单定稿后、披露之前 |
| 独立性 | 不要求 | 强制要求 —— 帮你编清单的顾问不能核查自己的成果 |
| 费用构成 | 内部工时 + 数据收集 | 核查人日：边界规模、抽样场所数、保证等级 |

保证等级分两档：**有限保证**（核查方声明"未发现存在重大错报的迹象"，
抽样为主、成本较低）和**合理保证**（正面声明清单实质准确，测试更深，
接近财务审计强度）。

## 核查员到底查什么

提前知道清单会被怎么查，是通过核查最便宜的方式：

- **边界完整性** —— 法人与场所有没有漏，排除项有没有成文理由。
- **数据链抽样** —— 抽若干条数据，要求出示原始凭证：电费单、燃料发票、
  制冷剂加注维保单。
- **排放因子适用性** —— 每个因子的出处、年份、地域是否匹配；GWP 版本
  是否一致。
- **复算** —— 抽样重算，期望落在你报告的数字上。
- **基准年与方法一致性** —— 变更是否说明、是否过了重要性检验。

## 什么情况下核查是强制的

- **碳市场 / MRV 体系** —— 全国碳市场重点排放单位（发电，以及扩容后的
  钢铁、水泥、电解铝等行业）每年接受主管部门组织的第三方核查；欧盟 ETS
  同理。
- **CSRD** —— 欧盟可持续报告指令要求对可持续声明做有限保证鉴证。
- **加州 SB 253** —— 在加州经营的大型企业范围一/二披露分阶段引入鉴证。
- **体系与客户要求** —— CDP 评分对经核证数据加分；大型采购方越来越多要求
  供应商提供经核查的清单；台湾对上市柜公司的强制盘查也配套了分阶段的
  查证时程。

都不适用时，核查是可选项 —— 本质是买公信力，在数字第一次对外公开那年
最值。

## 怎么编一份经得起核查的清单

核查之所以贵，多半是因为清单当初没按"会被检查"来编。差别就在**证据链**：
每个报告数字都能回溯到原始凭证、活动数据、指名道姓的排放因子和可复现的
计算。具体做法：凭证（账单、发票、抄表记录）跟数据存在一起，而不是躺在
某人的邮箱里；每一行都记录因子出处和年份；估算和缺口当场成文，不要事后
补写。从第一天就按这个纪律建账 —— 完整步骤见
[ISO 14064-1 清单编制七步](/zh/guides/ghg-inventory/) —— 核查就会从
"考古工程"变成"抽样练习"。
